## Install template

npx
```sh
yarn install
npx create-react-app my-app --template backoffice
```

npm
```sh
npm init react-app my-app --template backoffice
```

yarn
```sh
yarn create react-app my-app --template backoffice
```

## Usage

```sh
cd my-app
yarn install
yarn start
```
